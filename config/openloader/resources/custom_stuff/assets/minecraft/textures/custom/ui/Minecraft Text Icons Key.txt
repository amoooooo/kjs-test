To install this resource pack, copy the zipped folder into your resoure packs folder in your minecraft install folder.

To use this key, press CTRL+F to find the desired block/item/entity. Once found, copy the corresponding '' character that's found inbetween the '<>' characters. Paste into your chatbox in-game and, with have the resource pack installed, the icon will appear.

This resource pack uses the first 2240 characters of the Private Use Area (PUA). Given that, this pack is compatible with all custom fonts packs (unless they also use the PUA). There are another 4160 characters remaining in the PUA if you wish to expand.
